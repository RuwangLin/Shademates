# Contributing

ShadeMates is published primarily as a portfolio and evaluation snapshot of a completed team capstone. Issues that improve documentation, remove accidental sensitive information, or make the public build safer are welcome.

Before proposing a code change:

1. Do not include API keys, internal URLs, personal data, APKs, keystores, or university-internal documents.
2. Keep the five-feature final product scope and existing architecture understandable.
3. Add or update tests for behaviour changes.
4. Run `.\gradlew.bat testDebugUnitTest` and, when configured, `.\gradlew.bat assembleDebug`.
5. Preserve team attribution and the no-licence status unless all relevant rights holders agree to a change.

For security issues, follow [docs/SECURITY.md](docs/SECURITY.md) and do not disclose sensitive details in a public issue.
